## **Keyboard Controls**

| Key             | Action                  |
|-----------------|-------------------------|
| `W`            | Move forward            |
| `S`            | Move backward           |
| `A`            | Move left               |
| `D`            | Move right              |
| `SPACE`        | Ascend                  |
| `L-CONTROL`    | Descend                 |
| `LEFT ARROW`   | Rotate counterclockwise |
| `RIGHT ARROW`  | Rotate clockwise        |
